订阅clash梯子教程

订阅之前请先仔细阅读最终用户许可协议 

我这里讲的是如何订阅梯子，如果说你是想自己花时间搭建可以无需阅读这个教程。文章于2021年8月4号撰写或更新,教程仅限于已购买订阅用户使用。

获取应用

Clash for Windows 软件版权及后续权益属于软件作者，软件相关介绍及说明可参考作者主页：https://github.com/Fndroid/clash_for_windows_pkg

Clash for Windows 最新版本同时支持Windows和Mac平台，请确保您已安装最新版的Clash for Windows

先决条件

安装前请先检查一下先决条件：

  • 您拥有电脑的管理员权限
  • 您拥有dotnet runtime
  • 您拥有python解释器并将其添加到环境变量

这几个先决条件乍一看可能比较离谱，你可能会问这尼玛是什么玩意？what's up!这个翻墙就这么难？？？？我容易吗。。。别急，其实都很简单，对于第一个您拥有电脑的管理员权限，您只要是电脑的所有者且无第二人员来控制你的电脑你就会有管理员权限，说白了就是你只要安装软件到C盘就必须要有管理员权限。

对于检查第二个先决条件就略有些难度，但是依然很简单，检查的方法就是按下电脑上的win（即徽标键，就是一个Windows的logo）键和R键（两个键要同时按下）会弹出如下的界面，并在内输入cmd，就像这样：

会弹出一个黑漆漆的窗口

在内键入dotnet并回车，如果提示的是“dotnet不是内部或外部命令，也不是可运行的程序或批处理文件。”那么您需要到微软的官网下载runtime。如果是这样，则说明您拥有了dotnet，恭喜！

Microsoft .NET Runtime下载地址:https://dotnet.microsoft.com/download

下载好了可以自己验证一下自己有没有安装成功

对于第三个条件，我们可以通过下载python解决，python官网:www.python.org，下载好了之后安装不要玩了点击下面的Add Python to Path！

以上便是关于先决条件的所有解决方案

安装Clash

请先获取软件，链接在上方。打开链接后找到Releases，点击最新版本，会跳转到一个网页，点击以exe结尾的文件来下载clash

下载这一项，版本号任意

安装时请选择 “为使用这台电脑的任何人安装”；运行时请右键，以管理员身份运行

将你获取到的软件安装完毕后打开，并开启Start With Windows

点击Profiles，将你的订阅信息复制到这里，然后点击download，您的梯子就注册完成了

最后一步，回到general，开启System Proxy，可以愉快的科学上网啦
